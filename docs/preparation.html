1.1. エクスプローラーで見えるファイルの設定変更¶

  1. Windows + E キーでエクスプローラーを起動

    • インターネットエクスプローラーでは ありません

  2. 表示タブ をクリック ①

  3. オプション をクリック ②

  1. ◎ 隠しファイル、隠しフォルダー、および隠しドライブを表示する ③

  2. 下へドラッグして、④

  3. □ 登録している拡張子を表示しない ⑤

  4. OK をクリック ⑥
