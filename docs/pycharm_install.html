2. PyCharm と Anaconda のインストール¶

2.1. PyCharm インストール¶

2.1.1. JetBrains ToolBox のインストール¶

さまざまな JetBrains 社製の IDE 1 を簡単にインストール、アップデートできます。

  1. Jet Brains Toolbox App をダウンロードし

  2. ダウンロードしたファイルを実行します

1

IDE (Integrated Development Environment) は、ソフトウェアの統合的な開発環境であり、様々なツールの集合からなる → 統合開発環境

2.1.2. PyCharm Professional のインストール¶

  1. タスクバーのJetBrains Toolbox 立方体アイコン①をクリックするとメニューが上に広がります

  2. PyCham Professional を Install します

注釈

PyCharm Professional はひと月のお試し期間後に有料となります。学生ならば無料となります（ac.jp ドメインのメールアドレスを持っていること）
PyCharm Commnunity は誰でも無料で使えますが Anaconda 連携、Jupyter Notebook 連携など機能が削られています

  1. PyCharm Professional ダウンロード中の画面です。ダウンロードが終わったら、このままインストール中であることを表示します

2.2. Anaconda 2 のインストール¶

2

Anaconda(アナコンダ)とは、「データサイエンス」や「機械学習関連アプリケーション」のためのPythonおよびR言語用のディストリビューションです → オープンソースのAnacondaとは

2.2.1. ダウンロード¶

  1. Individual Edition

  2. Download をクリック

  1. 64-Bit Graphical Installer をクリックするとダウンロードが始まる

2.2.2. インストール¶

  1. ダウンロード (2.2.1 章) でダウンロードしたファイルを実行

  2. Next

  1. I Agree

  1. Just Me (recommended) をクリック ①

  2. Next ②

  1. Browse

  1. フォルダーの参照で ▷自分の名前 (ここでは Takashi YAMAMOTO）の左 ▷ ③ をクリックして下の階層を広げる（広がると ▽ になる）

  2. 下の階層が開いたら下へスクロール ④

    1. でスクロールすると、 AppData が現れるはず。 ▷AppData の ▷ ⑤ をクリックして下の階層を広げる（広がると ▽ になる、⑤ はすでに広がっている状態）

注釈

AppData が見えないときは、 エクスプローラーで見えるファイルの設定変更 (1.1 章)の設定がうまくいっていません。

  1. AppData のすぐ下 ▷Local の ▷ ⑥ をクリックして下の階層を広げる（広がると ▽ になる、⑥ はすでに広がっている状態）

  2. 下へ Programs が見つかるまで下へスクロールさせる ⑦

  3. Programs が見つかったらクリック ⑧

  4. 新しいフォルダーの作成 をクリック ⑨

  1. 新しいフォルダー が現れます、 どこもクリックせず

  1. 新しいフォルダー の上から小文字で anaconda3 と入力し⑩ ⏎

  2. anaconda3 ディレクトリができます

  1. OK で完了⑪

  2. C:\Users\あなたの名前\AppData\Local\Programs\anaconda3 と入力されていることを確認します（ここでは あなたの名前 が 16kcs）

注釈

うまくいっていないときは 7. からやりなおし。

  1. Next ⑬

  2. □ Add Anaconda3 to my PATH environment variable ⑭ （チェック外す）

  3. ☑ Register Anaconda3 as my default Python 3.8 ⑮ （チェック）

  4. Install ⑯

  1. インストールが終わったら Next （5 分程度お待ちください）

  1. Next ここで説明されているように PyCham Pro は Anaconda 推奨なのです

  1. □ Anaconda Individual Edition Tutorial ⑰ （チェック外す）

  2. □ Getting Started with Anaconda ⑱ （チェック外す）

  3. Finish ⑲

2.2.3. アップデート¶

ダウンロード （ 2.2.1 章 ）では十把ひとからげにいろいろ入っている最新のインストールパッケージをダウンロードしましたが、当然、インストールパッケージが作られた時以降のアップデートは反映されていませんので、ここでインストールパッケージが作られた時刻以降のアップデートを行います。

  1. スタートメニュー①を出します

  2. スタートメニューは ガン無視 して、 anaconda p と入力します②

  3. Anaconda Prompt (anaconda3) をクリックします③

  1. 黒い画面がでてきます（ 16kcs はご自身の名前になります）

  1. conda activate base⏎ ④

  2. conda update⏎ ⑤

  1. conda update --prefix から anaconda まで左クリックしながらドラッグして白く反転させます

7. 二回右クリックすると、ペーストされます

  1. ⏎ でアップデートを開始します

  2. 数分待つと何がアップデートされるのかごちゃごちゃでて、やっちゃっていいのか確認を求められます Proceed ([y]/n)? y⏎

  1. 途中経過が延々でてきます。長くて 10 分程度待ちます

  2. done とでで ⑥ 入力待ちとなるので、 exit⏎ で黒い画面を終了します ⑦
